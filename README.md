# Tomato_Crop_Health

The goal of this analysis is to see how soil aggregation metrics like water sieving are affected by cover crops and AMF in both Fusarium and non Fusarium infected crops. For this we will use a split plot design to analyze the data. We have multiple aggregation metrics but we will focus on WIN and use that as our response. We have used a split plot design and the data was collected through a completely randomized experiment. The analysis is in the files titled split_plot.

This was a project from when I worked at Statistical Consulting Service at the Ohio State University. All of the statistical and data analysis was done by me and I took into consideration the regular feedback from my client.

The secondary goal was to analyze the spectral data collected through satellites and find whether these single or multiple spectral bands can be used to predict the Carbon and Nitrogen levels of the crop based on their color. Due to some data consistency issues, I stopped at exploratory analysis of the data. This can be found in the file titled spectral_analysis.

A final report was written highlighting the major findings and next steps for this project in the file titled agriculture_report.
